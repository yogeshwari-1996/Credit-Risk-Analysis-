# Loan Risk Prediction & Business Analysis

## Project Overview

This project analyzes loan approval patterns using SQL and Python in Jupyter Notebook.

The objective is to identify factors affecting loan approvals and generate business insights for risk assessment.

---

## Tools & Technologies

- Python
- SQL
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SQLite

---

## Business Objectives

- Analyze loan approval trends
- Identify high-risk customers
- Study impact of income and credit score
- Generate KPI metrics for business reporting

---

## SQL Analysis Performed

- Average income analysis
- Approval rate by education
- Loan approvals by city
- High-risk customer identification
- Employment-based approval analysis

---

## Machine Learning

Built a Logistic Regression model to predict loan approval outcomes.

---

## Key Insights

- Higher credit scores improve approval probability
- Salaried employees have better approval rates
- Lower loan amounts are approved more frequently
- Income significantly impacts approval decisions
